# Xanthous Set

The bizarre yellow armor of the legendary exile King Jeremiah.

## Description

Mysterious yellow attire worn by Xanthous King Jeremiah. The crown is said to have been made in a lost land, and features a strange design.

## Set Pieces

- **Xanthous Crown**: 3.0 weight, 0 poise
- **Xanthous Overcoat**: 2.3 weight, 0 poise
- **Xanthous Gloves**: 0.9 weight, 0 poise
- **Xanthous Waistcloth**: 0.9 weight, 0 poise
