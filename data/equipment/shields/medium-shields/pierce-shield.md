# Pierce Shield

## Description

Employed by certain knights of Catarina, this shield features a large protrusion in the center for attacking. It can be used as a thrust weapon, offering both offensive and defensive capabilities.

## Stats

| Stat | Value |
|------|-------|
| Physical Defense | 95% |
| Magic Defense | 30% |
| Fire Defense | 70% |
| Lightning Defense | 50% |
| Stability | 53 |
| Durability | 180 |
| Weight | 3.2 |
| Strength Requirement | 10 |
| Dexterity Requirement | 10 |

## Special Properties

- Can perform shield bash attacks that deal thrust damage
- Shield attacks scale with dexterity

## Location

- **Gift from Siegmeyer of Catarina** in Blighttown swamp
- Give Siegmeyer 3 Purple Moss Clumps when you find him near the slugs
- Siegmeyer appears in Blighttown after you help him in Anor Londo

## Notes

- Unique ability to attack while blocking
- Shield bash has good range for a shield attack
- Does not provide 100% physical block
- Requires both strength and dexterity to wield effectively
