# Tower Shield

## Description
This metal greatshield was wielded by Knight Berenike, famed for his formidable armor. Constructed from iron, it provides exceptional defense but is exceedingly heavy.

## Stats

| Stat | Value |
|------|-------|
| Physical Defense | 100.0 |
| Magic Defense | 40.0 |
| Fire Defense | 75.0 |
| Lightning Defense | 50.0 |
| Stability | 86 (at +15) |
| Durability | 300 |
| Weight | 13.0 |
| Strength Requirement | 30 |

## Special Properties
- 100% physical block
- Crystal +5 gives highest stability in game (90)
- Good stability to weight ratio

## Location
- Dropped by Tower Knights in Undead Parish, Sen's Fortress, and Painted World of Ariamis
- Sold by Crestfallen Merchant for 8,000 souls

## Notes
- One of the most accessible greatshields in the game
- Best stability-to-weight ratio with 100% physical block
- Can be obtained early in Undead Parish
- Crystal upgrade path gives 90 stability but only 30 durability
