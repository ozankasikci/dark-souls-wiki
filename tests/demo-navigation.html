Navigation Enhancements Demo

This page demonstrates the new navigation features added to the Dark Souls Wiki.

1. Breadcrumb Navigation

Shows the current page hierarchy for easy navigation:

  1. Home
  2. Weapons
  3. Zweihander
Breadcrumbs appear automatically when viewing category listings or individual items.

2. Previous/Next Navigation

Navigate between items in the same category:

Previous Uchigatana Next Black Knight Sword
Appears at the bottom of item pages for easy browsing through related content.

3. Category Filters

Sort and filter items within categories:

Available on category listing pages. Different categories have appropriate filter options.

4. Table of Contents

For long articles with multiple sections:

Contents

  • Overview
  • How to Get
    • Location
    • Requirements
  • Statistics
  • Upgrade Path
  • Strategy Guide
Automatically generated for content with multiple headings. Provides quick navigation within long articles.

5. Features Summary

  • ✓ Breadcrumb navigation shows your current location
  • ✓ Previous/Next buttons for sequential browsing
  • ✓ Sort items by name, damage, soul reward, or recent additions
  • ✓ Filter items by type (weapons, armor, consumables, etc.)
  • ✓ Table of contents for long articles
  • ✓ Smooth scrolling to sections
  • ✓ Mobile responsive design
  • ✓ Dark Souls themed styling
